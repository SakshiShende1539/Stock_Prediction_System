## Project Overview

This project implements an end-to-end AI system for stock price sequence prediction using LSTM (Long Short-Term Memory) neural networks.

**What it does:**
- Downloads real stock data (Reliance Industries — NSE India) via Yahoo Finance API
- Trains a 3-layer stacked LSTM to predict the next day's closing price using the past 60 days
- Serves predictions via a FastAPI `/predict` endpoint
- Sends automated WhatsApp alerts via n8n + Twilio when a BUY signal is detected

**Architecture:**

```
    Yahoo Finance API
           │
           ▼
    Standalone Streamlit App (app.py)
    (Includes LSTM Brain + UI)
           │
           ▼
    Deployed to Streamlit Cloud 🚀
```

| Field | Details |
|-------|---------|
| **Dataset Name** | Yahoo Finance Historical Stock Data |
| **API Used** | `yfinance` Python library |
| **Source Link** | https://finance.yahoo.com |
| **Stock Ticker** | `RELIANCE.NS` (Reliance Industries, NSE India) |
| **Date Range** | 2018-01-01 to present |
| **Features** | Open, High, Low, Close, Volume |
| **Total Rows** | ~1,600 trading days |

### Preprocessing Steps
1. Download OHLCV data using `yf.download()`
2. Flatten multi-index columns (yfinance quirk)
3. Drop rows with missing values
4. Normalize all 5 features to [0, 1] using `MinMaxScaler`
5. Create sliding windows of 60 timesteps → predict day 61's Close
6. Split 80% train / 20% test (time-ordered, no shuffling)
7. Reshape to `(samples, 60, 5)` for LSTM input

## LSTM Mathematical Model

An LSTM cell uses gating mechanisms to learn long-term dependencies in sequential data:

### Gates

**Forget Gate** — What to discard from cell memory:
```
f_t = σ(W_f · [h_{t-1}, x_t] + b_f)
```

**Input Gate** — What new information to store:
```
i_t = σ(W_i · [h_{t-1}, x_t] + b_i)
C̃_t = tanh(W_C · [h_{t-1}, x_t] + b_C)
```

**Cell State Update** — Combine forget + input:
```
C_t = f_t * C_{t-1} + i_t * C̃_t
```

**Output Gate** — What to output as hidden state:
```
o_t = σ(W_o · [h_{t-1}, x_t] + b_o)
h_t = o_t * tanh(C_t)
```

Where:
- `σ` = sigmoid activation (squashes to 0–1)
- `tanh` = hyperbolic tangent (squashes to −1 to 1)
- `*` = element-wise (Hadamard) multiplication
- `h_t` = hidden state (short-term memory)
- `C_t` = cell state (long-term memory)

### How Sequence Learning Works
Each day's data `x_t` is passed through the LSTM cell along with the previous day's hidden state `h_{t-1}`. Over 60 timesteps, the model learns which price patterns (trends, dips, spikes) are significant for predicting the next close.

## Setup & Installation

### Prerequisites
- Python 3.9+
- Node.js 18+ (for n8n)
- Twilio account (free) with WhatsApp sandbox enabled

### Step 1 — Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/lstm-stock-agent.git
cd lstm-stock-agent
```

### Step 2 — Create virtual environment
```bash
python3 -m venv venv
source venv/bin/activate        # macOS/Linux
# venv\Scripts\activate         # Windows
```

### Step 3 — Install Python dependencies
```bash
pip install -r requirements.txt
```

### Step 4 — Train the model (Google Colab)
1. Open `LSTM_Stock_Prediction.ipynb` in [Google Colab](https://colab.research.google.com)
2. Run all cells (Runtime → Run all)
3. Download `lstm_model.h5` and `scaler.pkl` from the last cell
4. Place both files inside the `model/` folder in this repo

## Running the System

1. **Start the API Backend (Terminal 2):**


## n8n Agent Workflow

The n8n workflow automates stock monitoring and WhatsApp alerts:

```
Schedule Trigger (every 5 min)
        │
        ▼
  HTTP Request → GET /predict?symbol=RELIANCE
        │
        ▼
  Parse JSON response
        │
        ▼
  IF signal == "BUY"?
    ├── YES → Format WhatsApp message → Send via Twilio
    └── NO  → Log signal (no alert)
```

**Setup:** See [SETUP_GUIDE.md](./SETUP_GUIDE.md) for full step-by-step instructions including Twilio configuration.
